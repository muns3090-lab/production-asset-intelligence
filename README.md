\# 🎬 Production Asset Intelligence



An AI-powered asset request management system built for studio productions 

(Disney, WB, Sony-style environments).



\## Features

\- 📊 Real-time dashboard with asset status and category breakdowns

\- 🤖 AI triage: instant routing, risk flagging, and lead time estimates

\- 📋 Filterable asset request log by status, priority, and category

\- 💬 PAI Assistant: conversational AI for production operations queries



\## Tech Stack

Claude AI (Anthropic) · Streamlit · Python · Pandas · Plotly
